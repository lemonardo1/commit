# Features

번거로운 커밋 메시지, 이제 그만

코드 변경 사항을 분석해서 의미 있는 커밋 메시지를 즉시 생성합니다.

🤖

AI 자동 생성

git diff를 OpenAI GPT-4o 또는 Anthropic Claude에 전달해 정확한 커밋 메시지를 생성합니다.

📐

Conventional Commits

feat, fix, refactor, docs 등 표준 형식을 자동으로 지켜 팀 컨벤션을 유지합니다.

✏️

인터랙티브 편집

생성된 메시지를 바로 수락하거나, 직접 입력하거나, 에디터에서 편집할 수 있습니다.

🚀

원스텝 워크플로우

add → commit → push 전 과정을 하나의 명령어로 처리합니다.

🔒

안전한 API 키 저장

API 키는 ~/.commit/config에 600 권한으로 저장되어 외부에 노출되지 않습니다.

🐚

순수 Bash

별도 런타임 없이 bash, git, curl만으로 동작합니다. macOS와 Linux를 모두 지원합니다.

# Installation

3단계로 시작하기

설치부터 첫 번째 AI 커밋까지 1분이면 충분합니다.

1

설치 스크립트 실행

터미널에 아래 명령어를 붙여넣으세요. /usr/local/bin에 자동 설치되며, 권한이 없으면 ~/.local/bin에 설치됩니다.

curl -fsSL https://raw.githubusercontent.com/lemonardo1/commit/main/dist/install.sh | bash
2

API 키 설정

최초 1회 실행하면 AI 제공자를 선택하고 API 키를 입력하는 대화형 설정이 시작됩니다.

commit setup
3

커밋 시작

git 저장소에서 commit 명령어를 실행하면 AI가 변경 사항을 분석해 메시지를 생성합니다.

commit
# Usage

사용법

워크플로우에 맞게 다양한 옵션을 사용할 수 있습니다.

$ commit

AI 커밋 메시지 생성 → 확인 → push까지 한 번에

$ commit setup

AI 제공자와 API 키를 다시 설정

$ commit --help

사용 가능한 명령어와 옵션 보기

$ commit --version

현재 설치된 버전 확인

실행 순서

01 git add . — 모든 변경 파일 스테이징
02 git diff --cached를 AI에게 전달
03 생성된 메시지 확인 → Y 승인 / n 직접 입력 / e 에디터 편집
04 git commit -m "..." 실행
05 git push — 원격이 없으면 URL 입력 후 자동 추가

OpenAI

✓ bash 4.0+
